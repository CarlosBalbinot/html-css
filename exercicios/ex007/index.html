Alguns exemplos de formatação de textos

Negrito / Destaque

Nesta frase temos um termo em negrito usamos a tag b (Não semantica)

Nesta frase, tenho um termo em destaque usando a tag STRONG (semantica)

Itálico / Ênfase

Nesta frase temos um termo em negrito usamos a tag i (Não semantica)

Nesta frase, tenho um termo em destaque usando a tag em (semantica)

Marcador de texto

Podemos criar também um texto marcado usando a tag MARK

Também temos uma outra forma para trocar a cor do marcador do texto, utilizando direto o head

Texto grande e pequeno

Vamos criar um texto GRANDE e um texto pequeno nesta frase, utilizando tags.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado, utilizando a tag del

Podemos também realizar o riscado da frase ou da palavra desta maneira, utilizando a tag s

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, utilizando a tag ins

Podemos também realizar o sublinhado da frase ou da palavra desta maneira, utilizando a tag u

Texto sobrescrito

Para inserir coisas do tipo: x2+3. Utilizando a tag sup.

Texto Subscrito

Para inserir coisas do tipo: H2O. Utilizando a tag sub.
